# ✨ GlowAI – AI-Powered Skin Analysis Platform

GlowAI is a modern AI-powered skin analysis web application that helps users understand their skin health through intelligent facial image analysis and personalized skincare recommendations.

Users can upload a facial image and receive AI-generated insights about visible skin concerns, skincare suggestions, and personalized recommendations designed to support healthier skincare routines.

> ⚠️ Disclaimer: GlowAI is intended for informational and educational purposes only. It does not provide medical diagnoses and should not replace professional dermatological advice.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ABH137/glowai.git
```

Navigate to the project folder:

```bash
cd glowai
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
vercel dev
```

---

## 🔑 Environment Variables

Create the following environment variable in Vercel:

```env
OPENAI_API_KEY=your_openai_api_key
```

Do not expose API keys in frontend code.
